<h2>🚀 Overview</h2>

<p>
This project is a <b>Machine Learning-based healthcare web application</b> that predicts the likelihood of:
</p>

<ul>
  <li>🩸 Diabetes</li>
  <li>❤️ Heart Disease</li>
  <li>🧠 Parkinson’s Disease</li>
</ul>

<p>
Built using <b>Streamlit</b>, the app provides a simple UI where users can input medical parameters and instantly get predictions.
</p>

<hr>

<h2>✨ Features</h2>

<ul>
  <li>✔️ Multiple disease prediction in one platform</li>
  <li>✔️ User-friendly interface</li>
  <li>✔️ Fast ML predictions</li>
  <li>✔️ Lightweight and easy to deploy</li>
</ul>

<h2>📊 Dataset (Diabetes)</h2>

<p><b>PIMA Indians Diabetes Dataset</b></p>

<ul>
  <li>Total Samples: 768</li>
  <li>Features: 8</li>
  <li>Target: 0 (Non-Diabetic), 1 (Diabetic)</li>
</ul>

<hr>

<h2>🤖 Model Details</h2>

<ul>
  <li><b>Algorithm:</b> Support Vector Machine (SVM)</li>
  <li><b>Kernel:</b> Linear</li>
  <li><b>Training Accuracy:</b> 78.33%</li>
  <li><b>Testing Accuracy:</b> 77.27%</li>
</ul>

<h2>⚙️ Installation</h2>

<h3>1️⃣ Clone Repository</h3>

<pre>
git clone https://github.com/your-username/multiple-disease-prediction.git
cd multiple-disease-prediction
</pre>

<h3>2️⃣ Install Dependencies</h3>

<pre>
pip install -r requirements.txt
</pre>

<h3>3️⃣ Run App</h3>

<pre>
streamlit run main.py
</pre>
